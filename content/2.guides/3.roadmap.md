## Agent

- **Agent runner & tools** — Multi-turn sessions, tool use, streaming, and Pi coding-agent runtime. See the agent repo: `agent/docs/roadmap-agent-runner.md`.
- **Skills** — Documentation-driven skills, registry install, publish to store. See `agent/docs/skills-authoring.md` and the in-repo skills roadmap in `agent/docs/private/skills-roadmap.md` (if available).

## Integrations

- **Integrations roadmap** — Which provider tools exist and what to add next (GitHub, Gmail, Slack, Notion, Calendar, Linear, Discord, Airtable, Zoom, Jira, Stripe, etc.). See **`integrations/INTEGRATIONS_ROADMAP.md`** in the repo.

## Store

- **Store roadmap** — Registry contract, storefront, submit/admin, creator accounts, paid skills (Stripe Connect), sync from repo. See **`store/ROADMAP.md`** in the repo.
